OMV5的基本设置

发表于 2020-12-09 Valine：

”系统—-常规设置“ 里可以修改网页端口和管理员密码

建议更改下默认密码

这里插一句：每次更改设置后需要根据页面上的提示点击”应用“

然后需要在”更新管理“选项检查下更新，选择更新后点击升级下软件包

如果遇到更新失败可以少选几个再更新

点击插件选项页，选择”上传“，然后到这个页面下载OMV-Extras插件包，因为我装的是OMV5，所以下载这个

  • Download the plugin to enable this repository here:
    • For OMV 2.x (stoneburner) - https://github.com/OpenMediaVa…vextrasorg_latest_all.deb
    • For OMV 3.x (erasmus) - https://github.com/OpenMediaVa…extrasorg_latest_all3.deb
    • For OMV 4.x (arrakis) - https://github.com/OpenMediaVa…extrasorg_latest_all4.deb
    • For OMV 5.x (usul) - https://github.com/OpenMediaVa…extrasorg_latest_all5.deb
  • Go to the plugin tab in OMV’s web interface.
  • Upload the file.
  • Select the newly uploaded plugin openmediavault-omvextrasorg
  • Click on Install.
  • Refresh page.
  • Go to the plugin tab and click Check.

omv-extras网站上没有这个版本

装这个插件包的主要目的是为了安装docker

上传后检查更新，选择安装

这个最有用

安装好后到菜单栏就会和我一样再左侧多处OMV-Extras选项，点开这个，安装docker和Portainer

然后就可以图形页面管理Docker了

然后接着往下来设置共享文件夹，左侧选择”文件系统“—-”创建“—-设备选择需要格式化的硬盘，我这已经创建好了

看好硬盘，别选错了

接着选择”共享文件夹“—–”添加”，如下图所示创建共享文件夹，设备选择刚刚创建的硬盘

权限要设置好

点击”用户“选择需要设置的账户—-”编辑“—-”用户组“—-勾选”ssh“保存

勾选后可以通过远程命令访问，后续要用到

再点击”特权“，为用户分配刚刚创建的共享文件夹的权限

一般都设置读写

原谅我的不耐烦后面这些服务我就不写开启过程了，自己根据需求设置

这些个怎么设置网上都能搜索到
